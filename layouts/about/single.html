// pourquoi_ce_blog

La cloud security offensive est une niche quasi-absente du contenu FR. Les rares ressources existantes sont en anglais, niveau expert, et ne montrent jamais les erreurs ni les blocages.

Ce blog, c'est celui que j'aurais voulu avoir quand j'ai commencé : write-ups CloudGoat complets, fausses pistes incluses, remédiation concrète. Pour quelqu'un qui apprend, pas pour quelqu'un qui sait déjà. Les write-ups sont là.
